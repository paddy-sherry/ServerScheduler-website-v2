How does Server Scheduler work?

Connect Server Scheduler to your cloud account and you can then define when actions can be performed on a schedule. When the time comes, we execute the action. A typical example is turning off or downsizing a staging server overnight

How can I get started?

Click the start scheduling button at the top of this page and connect your cloud account.

What clouds are supported?

We integrate with AWS currently. Google Cloud and Azure are coming soon.

Is any scripting needed?

No. Server Scheduler is a point & click tool. No need to write scripts or terraform. Some people frown upon click ops but we like to keep things simple.
